how to install:
1. open this link to download python 13.0 https://www.python.org/ftp/python/3.13.0/python-3.13.0-amd64.exe
2. make sure to during installation to enable the checkmark "add python to path"
3. after installing open cmd as administrator and paste:"pip install pygame"
4. then into cmd as administrator paste:"pip install requests"+
5. open Brang.py with python to start

    2. find the "scale" toggler and switch it to 100%
    3. test again if it is working now
