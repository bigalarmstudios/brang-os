How to install:
1. Open this link to download Python 3.13.0: https://www.python.org/ftp/python/3.13.0/python-3.13.0-amd64.exe
2. During installation, enable the "Add Python to PATH" checkbox.
3. After installing, open an elevated command prompt and run: "pip install pygame"
4. Then run: "pip install requests"
5. Run Brang.py with Python to start

INFO: Please note that this is experimental.

If the window is not showing correctly, do this:
    1. Press Win + i and go to "Display"
    2. Find the "Scale" setting and switch it to 100%
    3. Test again to see if it is working now
